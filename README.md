# Portfolio Blazor

Portfolio personnel développé en C# / Blazor avec déploiement Azure.

Le projet met en avant :
- mes expériences professionnelles
- mes projets personnels
- un formulaire de contact connecté à un service email réel

## Technologies

- C#
- .NET 10
- Blazor
- Razor Components
- Azure App Service
- Resend API
- GitHub Actions

## Architecture

Le projet est structuré autour de :
- Models
- Components Razor
- Services
- Repository Pattern simple
- Injection de dépendances

  ## Fonctionnalités

- Affichage dynamique des expériences
- Cartes projets réutilisables
- Ligne décorative animée -- en cours
- Formulaire de contact interactif
- Validation des données
- Envoi d'emails via Resend

## À propos du développement

Ce projet a été conçu et piloté personnellement :
- choix d’architecture
- organisation des composants
- structure des modèles
- UX générale
- intégration des fonctionnalités

Certaines parties de code et phases d’implémentation ont été réalisées avec assistance IA afin d'accélérer le développement et d'explorer des pratiques modernes de l'écosystème .NET.
